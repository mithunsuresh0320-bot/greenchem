# 🌱 Green Chemistry AI

This project predicts whether a chemical is eco-friendly using Machine Learning.

## 🚀 Live App

[Click here to use the app](https://greenchemai.streamlit.app/)

## 🧪 Features

* Input chemical name or SMILES
* Converts name → SMILES automatically
* Predicts eco-friendly score
* Simple UI (Streamlit)

## 🛠 Tech Stack

* Python
* Scikit-learn
* Streamlit
* PubChemPy

## 📁 Files

* app.py → main app
* model.pkl → trained model
* vectorizer.pkl → feature extractor
